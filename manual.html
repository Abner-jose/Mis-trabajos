Manual Técnico del Sitio Web

1. Creación del Proyecto

Primero se creó una carpeta principal para almacenar todos los archivos del sitio web. Dentro de la carpeta se agregaron los archivos HTML, una carpeta llamada “imagenes”.

2. Diseño de la Página Principal

Se creó el archivo index.html que funciona como la página principal del sitio web. En esta página se utilizó un menú lateral y un iframe para mostrar el contenido de las diferentes páginas.

3. Creación del Menú

El menú contiene enlaces hacia:

  • Hoja de Vida
  • Notas
  • Galería
  • Video
  • Juego
  • Manual Técnico

Cada opción abre una página diferente dentro del iframe.

4. Hoja de Vida

Se creó una página con información personal del alumno, incluyendo nombre, carnet, carrera, habilidades y perfil.

5. Tabla de Notas

Se elaboró una tabla HTML que contiene los cursos actuales y las notas del primer y segundo parcial.

6. Galería de Imágenes

Se agregó una galería con 10 imágenes utilizando etiquetas HTML img.

7. Video de YouTube

Se insertó un video utilizando un iframe conectado directamente con YouTube.

8. Juego en Línea

Se agregó un botón que redirecciona hacia un juego ligero de motos en línea para cumplir con el requisito de JavaScript y entretenimiento interactivo.

9. Publicación del Sitio

Finalmente el sitio web fue subido a Internet utilizando un servicio de hosting gratuito para poder acceder desde cualquier dispositivo mediante URL.
